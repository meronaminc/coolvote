💡 요소 ① — 상단 안내 배너
노출 조건 투표 편집 화면 최초 진입 시 1회 자동 표시
숨김 조건 ✕ 버튼 클릭 또는 투표 시작 버튼 클릭 시 사라짐
재노출 한 번 닫으면 해당 세션에서 다시 표시하지 않음 (sessionStorage 활용 권장)
문구 투표 생성을 마치면 우측 상단의 투표 시작 버튼으로 투표를 시작하세요!
배경 linear-gradient(90deg, #eff6ff, #f0fdf4)
CTA 텍스트 ▶ 투표 시작 → (좌우 이동 애니메이션 반복)
애니메이션 CTA 텍스트: translateX 0 → 5px, 1.4s 무한 반복
✨ 요소 ② — 버튼 펄스 + 말풍선 툴팁
펄스 조건 배너가 표시된 동안 및 툴팁이 표시된 동안 버튼에 적용
펄스 종료 투표 시작 버튼 클릭 시 애니메이션 제거
펄스 스타일 box-shadow: 0 0 0 8px rgba(59,130,246,0)로 퍼지는 파동, 2s 무한 반복
툴팁 노출 배너를 ✕로 닫는 순간 버튼 아래에 말풍선 자동 표시
툴팁 숨김 ✕ 클릭 또는 투표 시작 버튼 클릭 시 사라짐
툴팁 문구 여기를 눌러 투표를 시작하세요!
툴팁 위치 버튼 바로 아래 (top: 100% + 10px, right: 0), 꼬리 방향 위쪽 오른쪽
툴팁 등장 opacity 0→1 + translateY(-6px→0) + scale(0.95→1), 0.35s
