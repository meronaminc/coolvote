투표 유형 선택 화면 — 유형별 설명 문구 추가 신규 추가
슬라이드 추가 시 표시되는 투표 유형 선택 팝업/패널
● 우선순위 보통
현재 투표 유형 선택 화면에는 유형 이름만 표시됩니다.
각 유형 이름 아래에 한 줄 설명 문구를 추가하여 선생님이 쉽게 이해하고 선택할 수 있도록 합니다.
유형 추가할 설명 문구 상태
☑️ 객관식 보기 중 하나 또는 여러 개를 선택하는 방식
⭕ 진위형 O/X 또는 True/False 중 하나를 선택하는 방식
📊 리커트 1~7점 내의 숫자로 의견을 표현하는 방식
🏆 순위 매기기 여러 항목에 순서를 매기는 방식
☁️ 워드클라우드 참여자가 단어를 입력하면 많이 나온 단어가 크게 표시되는 방식
✍️ 자유서술형 참여자가 자유롭게 문장을 작성하는 방식
디자인 가이드
· 설명 문구는 유형 이름 아래 보조 텍스트(12px, 회색 계열)로 표시해 주세요.
· 디자인 시안이 없는 경우 개발팀 재량으로 처리 후 기획팀 확인 요청 부탁드립니다.
